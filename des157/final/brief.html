comparison project 1

I am in absolute awe with the portfolio website designed by Kenta Toshikura. The website keeps most elements in a fixed position and changes the center elements as you scroll and drag. This doesn’t create a story necessarily, but it doesn’t need one because it makes you curious as each new object appears. I love how the pointer has a trailing dot and how it lights up the background oh so subtlely. The scrolling is so smooth and the numbers to the left let me know where I am. There’s even a toggle for accessibility viewing on the bottom left. It’s just great.
